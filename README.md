# Predicción de Rotación de Clientes (Churn)

**Objetivo:** Identificar factores de rotación y construir modelos predictivos.

**Modelos:** Regresión Logística (0.59 F1-Churn) y Random Forest (0.55 F1-Churn).

**Factores Clave:** Antigüedad, tipo de contrato, cargos mensuales, fibra óptica, cheques electrónicos, falta de seguridad/soporte técnico.

**Estrategias:** Incentivar contratos a largo plazo, enfocarse en nuevos usuarios, optimizar servicios y métodos de pago.
